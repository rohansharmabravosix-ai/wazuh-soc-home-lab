# wazuh-soc-home-lab
Wazuh-based SOC home lab built on Ubuntu and Windows 11 with Sysmon integration, endpoint monitoring, threat hunting, and security event analysis.
This project documents the deployment of a Wazuh-based Security Operations Center (SOC) home lab built using VMware, Ubuntu Server, and a Windows 11 endpoint.

The objective of the lab is to simulate a small-scale enterprise monitoring environment where security telemetry can be collected, analyzed, and investigated through the Wazuh Threat Hunting interface.

The lab includes Windows endpoint onboarding, Sysmon event collection, CIS benchmark monitoring, and security event analysis to demonstrate practical SIEM and SOC operations workflows.
